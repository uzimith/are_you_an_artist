# Are You an Artist

Illustlation Turn Base Game

お題にそって交代で一筆書きしていき、自分たち芸術家の中から一人の偽物を探しだすゲームです。ただしお題が偽物にバレると敗北です。

[Details](http://oinkgms.com/?pid=37395574)

## Prepare for development

### Server

- Node.js
- component
- Redis

### Client
- rubygems
  - compass
  - breakpoint

### Command

```
$ npm install
$ component install
$ gulp build
```
